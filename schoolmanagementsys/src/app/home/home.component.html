School Management System
Project overview - click to expand
Student Functionality
  • Add Student - Makes student entry in database
  • Delete Student - Deletes a student entry from the database
  • Update Student - Modifies student information in the database
  • View Student - Retrieves and displays student details from the database
Teacher Functionality
Create Teacher Profile

Allows administrators to create new teacher profiles in the system.

Delete Teacher Profile

Enables administrators to remove existing teacher profiles from the system.

Marks Management
Add Marks

Allows teachers to add marks for students for different subjects .

Edit Marks

Enables authorized users to edit the marks previously entered for students.

Delete Marks

Allows removal of marks for students, perhaps in cases of errors or adjustments.

View Marks

Provides a view of the marks entered for students, usually categorized by subjects or assessments.

Calculate Grades

Automatically calculates grades based on the marks entered for students according to predefined grading criteria.
